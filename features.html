01 / HTTPS Enforcement

ClickSafe uses Chrome's declarativeNetRequest API to silently upgrade HTTP connections to HTTPS before your browser even makes a request. You never navigate to an unencrypted page if a secure version exists — no interception, no slowdown, just a safer connection by default.

Network-layer redirect Zero latency overhead Mixed content flagged

02 / Tracker Blocking

ClickSafe loads 70,000+ known tracking domains on startup — covering advertising, analytics, social media, and fingerprinting networks. Every third-party hostname on a page is checked against this in-memory list in microseconds. No server call, no lookup fee, no delay.

70k+ domains O(1) lookup 24hr cache Disconnect.me list

03 / Link Safety

Before you click anything, ClickSafe runs a three-tier check — whitelist lookup, local SHA-256 hash prefix match, and only if there's a potential hit, a hash-only confirmation with the backend. Your actual URL never leaves the browser in normal operation.

SHA-256 local hash 30-min prefix refresh URL never transmitted Phishing · Malware

04 / Download Guard

Every file you download is intercepted the moment Chrome starts the transfer — before a single byte is written to your system. ClickSafe checks the download URL against Safe Browsing and cancels immediately if it's flagged. If the check can't complete, it fails safe.

Pre-disk interception Fails closed Safe Browsing API

05 / Dark Pattern Detection

ClickSafe scans every page for five types of manipulative UI — fake urgency, confirm shaming, countdown timers, pre-ticked checkboxes, and deceptive cookie banners. Detected elements are highlighted directly on the page so you see exactly what's trying to trick you, without any pop-ups or interruptions.

Fake urgency Confirm shaming Cookie manipulation Pre-ticked boxes

06 / Privacy Score

After every scan settles, ClickSafe computes a 0–100 privacy score for the page — deducting points for HTTP connections, tracking cookies, tracker scripts, and mixed content. The score drives the animated ring in the side panel and triggers a banner when a page falls below 50.
